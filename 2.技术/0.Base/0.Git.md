## git 操作

 下载/克隆远程仓库
git clone https://github.com/changmingxie/tcc-transaction.git

查看远程分支
git branch -a

查看本地分支
git branch

切换分支
git checkout aaa

显示远程地址
git remote -v

更新仓库
git pull
